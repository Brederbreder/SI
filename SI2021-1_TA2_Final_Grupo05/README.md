Para instalação do python e gerenciador de pacotes pip, 
execute em seu terminal:
$ sudo apt-get install python3-pip

Para instalação do framework Flask e seus módulos, execute a seguinte
sequência de comandos em seu terminal:
$ pip install flask
$ pip install flask-login
$ pip install flask-sqlalchemy

Para execução do programa, execute em seu terminal, dentro do diretório
raiz do trabalho:
$ python3 main.py

Assim, basta abrir seu endereço localhost, indicado no terminal, 
e navegar pelas funções do programa.
Ex: http://127.0.0.1:5000/
